# Vehicle Expense Calculator

A simple web application to calculate vehicle operating costs, reimbursement, and profit per mile.

## Features
- Calculate total miles driven
- Input reimbursement rate (e.g. $0.725 per mile)
- Estimate fuel cost based on MPG and fuel price
- Calculate total vehicle cost
- Determine profit per trip or day

## Use Case
Built for drivers who are paid per mile and want to understand their true profit after fuel and operating costs.
